# LMS-SBV — He thong Dao tao truc tuyen noi bo

Frontend React + TypeScript, kien truc tach UI/logic theo tung feature module.
13 module: auth, users, courses, learning, progress, quiz, question-bank,
departments, roles, reports, dashboard, settings, notifications.

## Cong nghe
React 18 + TypeScript + Vite, React Router, TanStack Query, Zustand,
Tailwind CSS, lucide-react, Recharts, Zod, Vitest.

## Cai dat va chay
npm install
Copy-Item .env.example .env
npm run dev
npm run build
npm run test
